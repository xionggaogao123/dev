[2015/12/21]
12月21日，由河南省基础教育教学研究室主办的“讲台炼名师”理化生同课异构活动在郑州隆重举行。来自湖北、安徽、湖南、河南四个省份九个地市的物理、化学和生物学科教学精英共聚郑州七中，向与会的数百位教育专家及学者呈献了一场精彩绝伦，风格各异的教学教研盛会。郑州七中副校长吉建州主持启动仪式，王保军校长致开幕词。河南省基础教育教学研究室李海龙主任做重要发言，倡导发挥名师在教师队伍中的辐射作用，促进教师间的切磋交流，营造教师探讨教学的学术氛围，打造名师队伍。
河南省基础教育教学研究室李海龙主任 化学组张春老师利用复兰科技平台进行信息化教学
专家教师纷纷上前仔细观看互动课堂教学
化学组老师张春运用复兰科技K6KT平台展示了一堂完美融合互联网+教育思维及翻转课堂教学方式的化学公开课，惊艳四座。微课视频在线推送、学生课前学习统计分析、学生问题反馈及实验展示、课堂快速答题统计等高科技的运用令在场专家教师目不暇接，纷纷从后座起身，凑近学生观看复兰科技互动课堂系统的强大功能，赞不绝口！
湖北省教育科学研究院的许晓林主任、河南省教研室刘志凤老师等数百位多名专家及一线教师现场听课，并做了精彩点评，突出强调因材施教的教学方式探究。合肥一中化学教研组长余志学老师期望我们的课堂能够更好的做到以人为本、以生为本，技术进课堂是未来课堂的发展趋势。
