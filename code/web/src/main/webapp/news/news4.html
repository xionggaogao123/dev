<返回新闻列表 浦东新区“校园台球联盟”
——体育活动实时分享
浦东新区是上海市台球运动的“领头羊”，在浦东唐镇——中国首个“台球之乡”的辐射力作用下，由浦东新区台球协会牵头，罗山中学、唐镇中学、唐镇小学、实验中学东校四校结成浦东新区“校园台球联盟”。联盟校将台球纳入体育课，开设台球特色课程，培养孩子兴趣爱好，并组建校台球队参加各类比赛，拓展视野，让学生拥有一个不一样的成长历程，满足广大家长对中小学生素质教育的热切期望。
联盟聘请职业教练指导培训，定期举行台球教学研讨活动、联盟论坛，加强校之间台球老师的观摩、交流、学习。复兰科技K6KT-快乐课堂平台帮助联盟校进行联盟资源建设，各校老师、学生学员可通过K6KT分享、查看专业台球视频教程资源，学习与提升台球技能。参赛选手和学生家长还可通过此平台观看联盟内台球联赛的视频直播，在线实时分享比赛精彩瞬间和活动感想。
浦东新区台球协会秘书长徐荣根表示：开展校园联赛、成立校园联盟，都是“台球要从娃娃抓起”理念的有益尝试，目前的四校联赛、四校联盟只是一个起点，未来将会扩展至8-10校。复兰科技K6KT-快乐课堂平台支持校园台球等体育活动的开展，为热爱、喜欢台球等体育活动的孩子们搭建好舞台，参与建设更加活跃的校园体育氛围，促进学生身心全面和谐发展。
