<返回新闻列表 复兰科技助力“长三角名校”齐聚黄山打造联盟资源
[2015/12/11]
2015年12月11日，“长三角”网络联盟校在黄山市成功召开，此次会议围绕校本课程资源和联盟课程资源的开发为主题展开，黄山市屯溪区教育局局长胡健红、副局长王峰等领导出席了会议并做了重要发言。参与此次会议的有安徽省黄山市屯溪江南实验小学、上海市杨浦区翔殷路小学、江苏省睢宁县实验小学、浙江省温州市龙湾实验小学等多所K12中小学名校。复兰科技作为学校的平台技术方为此次会议的顺利召开提供了全面支持。
网络联盟校一方面进行了兴趣课程经验交流，另一方面在校本课程研讨环节分别进行了互动课堂、二年级绘本课以及国学公开课程的展示和观摩。此次会议是皖沪苏浙四省市首度联手打造并分享基础教育优质教育资源，也开启了国内多地区联合打造教育资源的新篇章，为践行教育信息化、促进教育资源均衡，实现教育公平积累了宝贵的实践经验。
