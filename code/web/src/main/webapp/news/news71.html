<返回新闻列表 上海奉贤区智慧教育推进，复兰科技助力
[2015/12/30] [原文转自上海热线查看原文]
12月30日，由上海市奉贤区教育学院主办，肖塘中学承办的奉贤区初中学段智慧课堂展示活动拉开帷幕。奉贤区教育局副局长万国良、奉贤区教育学院院长蒋东标、奉贤区教师进修学院副院长金红卫、奉贤区教师进修学院教育培训管理中心副主任黄建龙、肖塘中学校长卜琴华出席了活动。
肖塘中学骨干教师杨彬和钱晓丽分别展示了《锐角的三角比》和《图形的运动》两节公开课。两位老师在常态化教学中坚持实践信息化技术应用，借助复兰科技K6KT-快乐课堂平台和互动课堂教学软件，联接课前预习、课中互动和课后复习，通过平台的统计分析功能实现个性化的分层教学。这次公开课向前来听课的领导和嘉宾们完整展示了信息化手段与一线教学的充分结合，让翻转教学、大数据统计分析、资源云存储、全向即时互动等高科技不再遥不可及，而变成课堂教学的常态。来自奉贤区教育学院及教师进修学院的专家、奉贤区各校的信息化分管领导、教导主任及优秀教师代表百余人观摩了公开课，并参与研讨评课。
万国良局长、蒋东标院长分别就信息技术与日常教学相结合分别提出了建设性意见。卜琴华校长介绍了肖塘中学信息化推进的进程，并表示在今后的的教学信息化建设中，将会继续借助技术和信息化手段，不断丰富学校教学资源，提升教学质量。最后，金红卫院长对此次活动给予高度评价，同时也提出了更高的要求，希望不仅是奉贤区教育学院的四大中心联合起来举办这样的活动，各所学校也要联合起来，针对不同的问题，寻找非常契合的解决问题的办法，提升教学质量，打造学校特色。
