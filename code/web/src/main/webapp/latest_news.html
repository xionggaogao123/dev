    海市晋元高级中学前身为创办于1904年的华童公学，为纪念爱国将领谢晋元将军而命名为“晋元中学”，目前为上海市市级重点中学，首批“上海市实验性示范性高中”。晋元中学实施“选择教育”的办学理念，坚持以学生发展为本，要求学生“学会选择、主动学习、卓越发展”，实行“套餐式”组合课程、“走班制”运作方式、“学分制”管理模式。
