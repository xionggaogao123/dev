-----“3+3走班选课”典范
上海市晋元高级中学前身为创办于1904年的华童公学，为纪念爱国将领谢晋元将军而命名为“晋元中学”，目前为上海市市级重点中学，首批“上海市实验性示范性高中”。晋元中学实施“选择教育”的办学理念，坚持以学生发展为本，要求学生“学会选择、主动学习、卓越发展”，实行“套餐式”组合课程、“走班制”运作方式、“学分制”管理模式。
晋元高级中学成功运用复兰科技K6KT-快乐课堂平台实行“3+3走班选课”。学生在线进行选课，平台根据学生人数、性别、特长等自动均衡编班，分别生成学生、任课老师及行政班班级课表，进行全校课表汇总，师生根据权限随时查看排课情况，形成科学评价体系。这样既可以满足学生个性化教育需求，达到因材施教的教育效果，同时将走班课程与非走班课程进行有效统一管理，大大减轻学校教务处走班排课繁杂的工作量，提高了教学管理的高效性和科学性。
湖北省教育科学研究院的许晓林主任、河南省教研室刘志凤老师等数百位多名专家及一线教师现场听课，并做了精彩点评，突出强调因材施教的教学方式探究。合肥一中化学教研组长余志学老师期望我们的课堂能够更好的做到以人为本、以生为本，技术进课堂是未来课堂的发展趋势。
